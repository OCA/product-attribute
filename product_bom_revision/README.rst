========================
Product BOM revision
========================

This module allows you to store the revision of a product and a BOM.

Usage
=====

* Go to Inventory > Master Data > Products
* Create or edit a product.
* Enter the Revision and save.
* Create or edit a BOM.
* Enter the Revision and save.
