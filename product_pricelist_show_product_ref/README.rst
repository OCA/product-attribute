This module allows you to view the full product name with the respective Internal Code on a 'Pricelist items' tree view.

**Table of contents**

.. contents::
   :local:

Configuration
=============

To configure this module, you need to:

#. Go to *Sales > Configuration > Settings* and check
   "Multiple Sales Prices per Product" option and
   "Prices computed from formulas" after that.

Usage
=====

To use this module, you need to:

#. Go to *Sales > Products > Pricelists* and create a new Price List.
#. Add a Pricelist Items to that Price List.
#. Save your changes.
#. Verify that the Product is displayed with the full name (including the Internal code)
